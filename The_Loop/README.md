## 💡 About This Project
 
This project was built as my first checkpoint assignment on [Codedex.io](https://www.codedex.io/home) while practicing Python. I wanted to build something I'd actually want to play, so I took inspiration from the *Goosebumps* choose-your-own-adventure books and wrote my own horror story from scratch. The name *The Loop* is a nod to both the story's structure and the `while` loops that power the game itself.
