# 🌀 The Loop
> *A terminal-based choose-your-own-adventure horror game built in Python*

Every morning you wake up and go through the same routine. But something feels different. And then something feels *wrong*. And then you can't ignore it anymore.

**The Loop** is a text-based horror adventure where your choices shape how the story unfolds — but no matter what you do, the morning always comes back around.

---

## 🕹️ How to Play

**Requirements:** Python 3.x

**To run the game:**
```bash
python the_loop.py
```

The game runs entirely in the terminal. When prompted, enter **1** or **2** to make your choice and advance the story. No installs or dependencies required — just Python and your nerve.

---

## 📖 Gameplay Overview

The story unfolds across several loops, each morning feeling a little more off than the last. Early on, your choices are low-stakes — stay home and game, go shopping, call a friend. But as the loops progress and strange things start happening, the decisions you make start to matter a lot more.

- 🌅 **Multiple loops** — each morning builds on the last
- 🔀 **Branching choices** — your decisions affect what happens next
- ⚠️ **Input validation** — the game won't let you break it with bad input
