## Projects

### 🌀 The Loop
> *A terminal-based adventure built in Python*

I have been using [Codedex.io](https://www.codedex.io/home) to practice Python and reached my first checkpoint assignment. I chose to do a text based game that uses the terminal to get responses from the user. Inspired by the Goosebumps choose-your-own-adventure books, I decided to take a stab at writing my own choose-your-own-adventure horror game. I named this project The Loop not only due to the story's nature. This project allowed me to practice writing while loops that connected to each other. It also helped me to practice try/except blocks as well as syntax. As I learn more I plan to revisit this project so that I can use my newly gained skills to improve the project's efficiency and readability. 

**Built with:** Python  
**Concepts practiced:** Control flow, functions, user input handling, game state management  
**Future Updates Planned:** Refactor repetitive while loops into a reusable function, revise and edit the game's story to provide more context
